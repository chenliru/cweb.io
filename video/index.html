Information

Web Video is built using VideoCapture, which provides an easier way to capture Media streams. Click the "Start" button to begin video recording for a few seconds. You can stop the video by clicking the creatively-named "Stop" button. The "Download" button will download the received data (although it's in a raw, unwrapped form that isn't very useful).
